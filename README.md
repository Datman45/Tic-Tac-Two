# Tic-Tac-Two

## How to install
- Clone or download project
- Open the project in Visual Studio Code
- Build and run project

## How to run project
- Write or copy in terminal
``` bash
npm run dev
```
- Control + left click on the local link
- Login menu will appear

## How to log in
- Write any username and password (minimum 1 character)
- Choose 1 game mode
- Click submit

## Game modes
- Human vs Human
  - Two players play on the same screen
  - Players take turn after each move
- Human vs AI
  - you play against AI
  - AI difficulty is fixed
    - Can make random moves
    - Can block opponent`s winning moves
    - Can make winning moves
- AI automatically moves after player`s turn

## Pages Overview
- Home Page
  - Displays a welcome message with the username
- User Info Page
  - Shows login data and selected game mode
- Game Page
  - Display game board
  - Main gameplay happen here
- Log Out Page
  - Allows to player to exit their profile

## How to start game
- Click on the "Game" in the navigation menu
- ⚠️ Important: Each player has 30 seconds to make a move, or the turn will automatically switch

## How to play game
- Click on any red cell to make a move
- Now other player`s turn

## Four Pieces Rule
- When player has 4 pieces on the board. Two options become available:
  - Move grid
    - Move entire grid one cell in any direction
  - Move piece
    - Move piece on any red empty cell

## Game end
- Game ends when:
  - One of the player or AI wins
  - You will be redirected to the Game Over page, where the winner is displayed.
